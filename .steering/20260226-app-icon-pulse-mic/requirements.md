# requirements.md

## 1. 目的
NYvoice のアプリアイコン案として採用した「Pulse Mic」を、ミニマム方針で実運用可能なラフセットへ精緻化する。

## 2. 背景
- 案1（Pulse Mic）と案4（Menu Dot）の比較を実施し、案1を採用した。
- 次工程では、アプリアイコン（カラー）とメニューバー表示（モノクロ template）に分けて設計する必要がある。

## 3. 対象範囲
### In Scope
- Pulse Mic の 1024px マスターラフを微調整
- 小サイズ検証向け 32px / 16px の簡略版ラフ作成
- メニューバー向けモノクロ template アイコンラフ作成
- 案の意図・使い分けを README に整理

### Out of Scope
- 最終的な `icns` 生成
- Xcode Asset Catalog への正式組み込み
- ブランドガイドライン策定

## 4. 機能要件
- FR-01: Pulse Mic の中心モチーフ（マイク + 単一パルス）を維持する
- FR-02: 16px / 32px で視認できるようにディテールを削減する
- FR-03: template 用は単色で可読性を優先する
- FR-04: すべて SVG で保存し、次工程の書き出し元として扱えること

## 5. 非機能要件
- NFR-01: ミニマムデザイン（形状要素は最小限）
- NFR-02: dark menu bar 上で視認できるコントラスト
- NFR-03: 既存ドキュメント構造に従って成果物を整理

## 6. 受け入れ基準
- 1024 / 32 / 16 / template の4種が作成されている
- README に用途と次工程が記載されている
- 採用案（Pulse Mic）として判断に必要な比較情報が揃っている
