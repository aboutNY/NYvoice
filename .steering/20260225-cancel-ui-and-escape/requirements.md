# requirements.md

## 1. 目的
無音キャンセル時のユーザー認知を改善し、開発リスト No.3（Escape キー終了）を実装する。

## 2. 要求
- 完全無音時に即座にUIを閉じず、キャンセル表示を短時間出してから閉じる
- 録音中に Escape キー押下で録音を終了し、無音時と同様のキャンセル表示を出して閉じる
- 通常の停止（Stop ボタン/ショートカット再押下）は従来どおり処理継続する

## 3. 受け入れ基準
- 無音録音終了時に「キャンセル」表示が出てからモーダルが閉じる
- 録音中 Escape で文字起こし/挿入を行わず、同じキャンセル表示後に閉じる
- 発話ありの通常停止は挿入まで実行される
