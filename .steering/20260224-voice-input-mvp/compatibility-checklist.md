# 10.3 Manual Compatibility Checklist

## 目的
主要アプリでカーソル位置挿入が実際に機能するかを確認する。

## 事前条件
- NYvoice が起動している
- Environment Check で重大エラーがない
- Whisper/Ollama の設定が有効

## 実施手順（各アプリ共通）
1. 対象アプリで入力欄にカーソルを置く
2. NYvoice の録音開始ショートカットを押す
3. 10秒程度話す
4. 停止操作を行う
5. 挿入結果を確認する

## 対象アプリ
- [ ] TextEdit
- [ ] VS Code / Xcode（エディタ入力）
- [ ] Slack / Discord（チャット入力）

## 確認観点
- [ ] テキストがカーソル位置に挿入される
- [ ] 既存テキストを不正に破壊しない
- [ ] 日本語文字の欠落/重複がない
- [ ] 権限不足時は適切なエラーメッセージが出る
