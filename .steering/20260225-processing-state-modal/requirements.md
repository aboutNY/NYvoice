# requirements.md

## 1. 目的
録音停止後から文字挿入完了までの待機時間に、ユーザーへ処理中であることを明示する。

## 2. 背景
現状は録音状態を抜けるとモーダルが閉じるため、処理中か異常停止か判別しづらい。

## 3. 要求
- `Transcribing/Correcting/Inserting` の間はモーダルを維持する
- 処理中表示（例: 「処理中です」）を表示する
- 文字挿入完了でモーダルを閉じる

## 4. 受け入れ基準
- 録音停止後にモーダルが閉じず、処理中表示に切り替わる
- 挿入完了でモーダルが閉じる
- 既存の録音停止操作は回帰しない
